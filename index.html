Recall It (React, Python)

Recall.it is the Google Chrome extension for alerting buyers from unsafe products on Amazon that could harm kids. It released in April of 2020. The project was built by a team of 4 developers, utilizing React and Python.

In December 2019 I started volunteering for the Kid. The Kid is a nonprofit organization dedicated to protecting children by fighting for product safety. I held front end developer position to build basic layouts, to fetch data, and render it to the page.

The entire work was done remotely, once a week we had meetings with the team. We worked according to Agile methodology, which was effective for all.

Recall.it is a great helper for you, it provides detailed information about the product like data about how many kids were injured from the current item, how hazard is it. You can learn more about the product, reading the "Consumer Reports" section.

Cinematica (React, Node JS, Express)

Visit website

Cinematica is an online database of information related to films that supposed to be played in park zones in the Chicago area. If you like watching movies, you will love to watch it outdoor in the park. Save movie sessions in your list, if you planning not to get wet or get burned on the sun, check the weather forecast provided in the app. For the strong user experience, I had rendered Google Map with the location where the current event being played. The project built utilizing React & Node.

The idea to build the project came from IMDB online database. The goal was to learn how to build RESTFUL web services, work with third-party APIS. Chicago park district provides great information about events happening in the city and luckily has API documentation about movies in parks.
